## Проделанная работа

Для каждой лабораторной работы был создан файл `CodeExplanation.md` в соответствующей папке. В этих файлах объясняется
работа кода, структура классов, методы и ключевые слова C++ (такие, как `explicit`, `const`, `virtual`), используемые в
контексте программы. Это помогает понять принципы объектно-ориентированного программирования, наследования, шаблонов и
перегрузки операторов.
