# Объяснение кода Lab4

## Обзор программы

Программа реализует шаблонный класс `Vector<T>` для работы с векторами произвольного типа. Перегружает операторы для
сложения, вычитания, скалярного произведения. Демонстрирует работу с векторами целых чисел и строк.

## Структура кода

- **Vector.h**: Шаблонный класс Vector с операторами.
- **main.cpp**: Демонстрация для int и std::string.

## Ключевые компоненты

### Класс Vector<T>

- **Поля**:
    - `T* data`: Массив элементов.
    - `size_t size`: Размер вектора.

- **Конструкторы**:
    - `Vector()`: Конструктор по умолчанию, делегирует к `Vector(size_t n)` с n=0, создает пустой вектор.
    - `explicit Vector(size_t n)`: Вектор размера n. `explicit` предотвращает неявное преобразование.
    - `Vector(const Vector& other)`: Копирование.

- **Деструктор**: Освобождает память.

- **Оператор присваивания**: Глубокое копирование.

- **Доступ**: `operator[]` с проверкой границ, константная версия.

- **Операторы**:
    - `operator+`: Поэлементное сложение.
    - `operator-`: Поэлементное вычитание.
    - `operator*`: Скалярное произведение (для чисел) или специальная операция для строк.

- **Метод**: `print() const`: Вывод вектора.

### Специальные операторы для std::string

- `operator-`: Удаление подстроки.
- `operator*`: Повторение строки.

## main.cpp

Создает векторы int и string, выполняет операции и выводит результаты.

## Ключевые слова и их значения

- **explicit**: В конструкторе `Vector(size_t n)` предотвращает неявное преобразование типов.
- **const**: В методах, не изменяющих объект (print, getSize, операторы). В параметрах.
- **template**: Определяет шаблонный класс, параметризованный типом T.
- **typename**: В шаблоне указывает, что T - тип.
- **virtual**: Не используется. Нет наследования.
- **inline**: Для операторов строк, определены в заголовке.

## Работа программы

Программа демонстрирует векторные операции для разных типов, показывая гибкость шаблонов.
