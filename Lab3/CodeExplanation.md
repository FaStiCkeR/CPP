# Объяснение кода Lab3

## Задача
  Создайте класс Строка с обязательными член–данными: длина строки, указатель на строку (строка хранится в динамически
выделенной памяти). Обязательные методы: конструктор по заданной длине строки, конструктор по заданной строке,
конструктор по другому объекту класса Строка, конструктор копирования, очистка строки, вывод строки. 
Перегрузить (переопределить) операции: + (сумма арифметическая ), ==(сравнение), <= (меньше либо равно), >= (больше либо равно).

## Обзор программы

Программа реализует собственный класс `String` для работы со строками, аналогичный `std::string`, но с ручным
управлением памятью. Демонстрирует конструкторы, операторы перегрузки, копирование и сравнение.

## Структура кода

- **String.h**: Заголовок с объявлением класса `String`.
- **String.cpp**: Реализация методов.
- **main.cpp**: Демонстрация использования класса.

## Ключевые компоненты

### Класс String

- **Поля**:
    - `size_t length`: Длина строки.
    - `char* data`: Указатель на массив символов.

- **Конструкторы**:
    - `String()`: Конструктор по умолчанию, делегирует к `String(size_t n)` с n=0, создает пустую строку.
    - `explicit String(size_t n)`: Создает строку из n пробелов. `explicit` предотвращает неявное преобразование.
    - `String(const char* s)`: Из C-строки.
    - `String(const String& other)`: Копирование.

- **Деструктор**: Освобождает память.

- **Оператор присваивания**: `String& operator=(const String& other)` - глубокое копирование.

- **Методы**:
    - `clear()`: Очищает строку.
    - `print() const`: Вывод строки.
    - `c_str() const`: Возвращает C-строку.

- **Перегруженные операторы**:
    - `operator+`: Конкатенация.
    - `operator==`: Равенство.
    - `operator<=`, `operator>=`: Сравнение лексикографическое.

## main.cpp

Создает несколько объектов String, демонстрирует операции: создание, копирование, конкатенацию, сравнение, очистку.

## Ключевые слова и их значения

- **explicit**: В конструкторе `String(size_t n)` предотвращает неявное преобразование, например, `String s = 5;` не
  скомпилируется.
- **const**: В методах, не изменяющих объект (print, c_str, операторы сравнения). В параметрах для защиты.
- **virtual**: Не используется. Нет наследования.
- **inline**: Не используется явно, но функции определены в заголовке для простоты.

## Работа программы

Программа создает строки, выполняет операции и выводит результаты, показывая функциональность класса String.
